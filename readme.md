# React/Redux boilerplate

## Dependencies: 
- React
- Redux
- Styled-Components

## Development dependencies:
- Webpack
- Code splitting
- ESLint
- Prettier
- React Hot Loader
- Flow
- Jest

## Getting started

```
// installation
git clone https://github.com/kme211/react-redux-boilerplate.git

// run dev server
npm start

// build
npm run build

// lint
npm run lint
// or
npm run lint:watch
```
